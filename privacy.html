WebView close request (stub)

The Close WebView button calls the function requestCloseWebView(). It sends a POST request to a local stub endpoint: /webview/close. Your app can intercept this request in WebView and close the screen.

If the endpoint is not handled, the request fails silently and shows a small toast — safe for GitHub Pages.
